### Active Workstreams

1. **Watercolor Stroke Painting** (`watercolor-chris.html`)
   - Stroke-based painting (petals as brush strokes, not stamps)
   - Tyler Hobbs polygon deformation + Mixbox pigment mixing
   - Progressive "watching paint dry" system
   - Status: looks good but needs more organic brush movement (see feedback in session notes)

2. **fatiha.brush Engine** (`fatiha-brush.js` + `fatiha-brush-demo.html`)
   - Custom WebGL instanced quad renderer, zero dependencies
   - 25K+ stamps at 60fps
   - Procedural SDF stamps in fragment shader
   - Status: functional but watercolor-chris.html approach looks better

3. **Tawaf Simulation** (`tawaf-sim.html`)
   - MiroFish-style swarm sim of pilgrims circling Ka'bah
   - Social Force Model (Helbing & Molnar 1995)
   - 6 demographic agent types, Bayesian crowd predictor
   - Real Mataf dimensions + capacity (107K/hr, sourced)
   - Touching layer, squeeze zone, Black Stone, Multazam
   - Prayer mode with circular rows + ruku/sujud poses
   - Energy vortex visualization
   - Live: https://cookmom.github.io/tawaf-sim/
   - Status: deployed, iterating

4. **Tawaf + AGOT Merge** (`tawaf-agot.html`)
   - AGOT glass cube scene with pilgrim particles orbiting in 3D
   - Status: first pass, particles too big, needs refinement

5. **Islamic Manuscript Slides** (`slide-tawaf.html`, `slide-light.html`, `slide-garden.html`)
   - Code as sacred mathematics on illuminated manuscript pages
   - Both CSS and Gemini NBP2 generated versions exist

- Dislikes: sharp computer-generated edges, desaturated pastels, translucent overlap confusion
