## Grain Texture Deep Dive — Late Night Session (v172→v177)

### The Journey
- v172-173: CSS grain overlays (body, per-section, canvas-sibling) — all workarounds
- v174-175: Material-based with random noise normal map — loaded correctly but invisible or ugly ("sandpaper")
- v176: Tuned random noise params — visible on phone but ugly
- **v177: Turing reaction-diffusion vermicular pattern — the real solution**

### Key Breakthrough: Turing Patterns, Not Random Noise
- Random noise normalMaps look like sandpaper (every pixel = random direction)
- Bauhaus grain is a **vermicular/labyrinthine pattern** — coherent flowing worm-like ridges
- Generated via **Gray-Scott reaction-diffusion** (F=0.055, k=0.062, 10000 iterations, periodic boundaries)
- Files: `turing-grain.png`, `turing-normal.png`, `turing-roughness.png` (all 512x512)
- Sobel-derived normal map with strength 2.5

### Research-Backed Material Parameters (v177)
- **metalness: 0.0** — galvanized dials are dielectric (painted/lacquered), NOT bare metal
- **roughness: 0.35** — semi-matte; too rough (>0.6) = grain disappears in diffuse; too smooth (<0.2) = mirror
- **normalScale: (0.5, 0.5)** — Turing patterns tolerate higher values than noise due to smooth gradients
- **clearcoat: 0.3, clearcoatRoughness: 0.4** — lacquer layer HELPS grain (smooth specular over rough base)
- **envMapIntensity: 1.0** — grain shows primarily in env reflections
- **repeat.set(4, 4)** — 512px tiled 4x on ~200px dial = grain reads as texture, not pattern
- **scene.environmentIntensity: 0.7**
- Raking directional light at (-100, 60, 80) intensity 1.2

## Grain Texture — RESOLVED (v210)
- **Final texture**: `grain-fine.png` = spectral clone (Bauhaus frequency profile + random phase + histogram matched)
- **Technique**: FFT magnitude from Bauhaus tile → random phase → inverse FFT → histogram match to Bauhaus contrast (std=77, range 8-251)
- **Result**: Same cellular vermiculate character as Bauhaus, completely new pixel arrangement (no IP issues), seamless tiling
- **CSS recipe**: `soft-light` @ `0.4`, `background-size: 148.5px auto`, tile size 198×198px
- **Key lesson**: `plus-lighter` is additive — washes out on mid-gray backgrounds. Only works on light backgrounds like Bauhaus's #f5f5f0. `soft-light` preserves base color and adds texture contrast symmetrically.
- **Key lesson**: NOMOS sandblast grain has too much low-frequency energy — reads as TV static, not surface texture. Bauhaus vermiculate has tight, uniform feature sizes that read as "material" not "noise".
- **Versions burned**: v201-v209 were iterations (too soft, too heavy, wrong blend mode). v210 is the one.
- Tawfeeq approved: "It looks amazing! I love it"
