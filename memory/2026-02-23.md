# 2026-02-23 — Glass Prism Cube Day

## Key Breakthroughs

### 1. ShaderMaterial vs Transmission FBO
**ROOT CAUSE FOUND:** Three.js's `MeshPhysicalMaterial` transmission FBO pre-pass ONLY captures objects using built-in materials (MeshBasicMaterial, MeshStandardMaterial, etc). Custom `ShaderMaterial` objects are INVISIBLE to the transmission FBO. This is why the glass cube appeared opaque — the ring meshes used ShaderMaterial and weren't captured.

**Fix:** Converted ring meshes from ShaderMaterial → MeshBasicMaterial with a compatibility shim for the color-setting code.

### 2. Proxy Dedup Fixes (5+ iterations)
- Session key was `undefined` → all requests shared one slot → heartbeat killed main session
- Fixed: session key from `body.user` > `X-Session-ID` header > model name
- Detached subprocesses sent stale responses → socket destroy instead of graceful close
- Completion cache by last-user-message hash to prevent retries from regenerating
