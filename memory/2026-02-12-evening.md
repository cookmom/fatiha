## Swap Lesson (CRITICAL)
- When replacing large JS arrays in HTML, orphaned code can remain after the array's `];` if the end-detection is wrong
- Backticks inside template literals confuse line-based bracket counting
- Always verify what's AFTER the replaced section — check `sed -n 'ENDLINE+1,+10p'`
- This session: 579 lines of orphaned code from bad first swap broke the entire page
