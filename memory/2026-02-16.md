### Key Technical Discoveries
- Font paths: `opentype.loadSync()` → `font.getPath()` → normalize to center → Three.js Shapes
- Arabic glyphs = independent stroke contours, NOT holes. Each contour extruded as separate shape.
- Kufi commands ~15-55 per glyph vs Lateef ~20-50 — simpler geometry, bolder presence
- `pathToShapes()` linearizes bezier curves for Shape compatibility (quadratic → 4 segments, cubic → 5)
