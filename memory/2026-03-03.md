### Key Bug Pattern Learned
- **ES module scope isolation**: Any function in a `type='module'` script is NOT global. Must explicitly `window.funcName = funcName` to expose. This bit us with `_swipeRevert` and `_displayPrayerTimes`. Both now on `window`.
