### Architecture decisions (v18)
- **Dot-presence-driven split**: `dotPresence[]` Float32Array marks which curve indices have living dots, strength fades as dots age. Gaussian blur (3 passes) + splitBlur radius (40 indices) for smooth edges
- **Dots stick to line**: Each dot stores `{ci, side, offX, offY}` instead of absolute `{x, y}`. Draw loop recalculates position from `curvePoints[ci] + normal * displacement * side + offset` every frame
- **Spring-damper rejected**: Elastic overshoot looked unnatural. v10's simple gaussian displacement + lerp smoothing preferred
