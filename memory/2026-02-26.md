### Key Technical Decisions
- **Polar disc > card geometry**: CircleGeometry + angular shader = no visible edges, looks like actual light
- **FRAG_PRISM_FAN shader**: Piecewise spectral interpolation (violet‚Üícyan‚Üígreen‚Üíyellow‚Üíorange‚Üíred), Gaussian radial falloff, caustic shimmer via `sin(r*40+time*0.5+theta*3)`
- **`cubeIntensity` in shader**: `0.3 + 0.7 * smoothstep(0.35, 0.06, r)` ‚Äî brighter near cube, sells "cast by cube"
- **3 disc layers**: Fan (r=12, op=0.55), bloom (r=14, op=0.22), entry beam (r=4, op=0.18)
- **Caustic PointLight** (0xff8844, intensity 0.5) under cube ‚Äî makes cube edges glow
- **`_compassLocked` flag**: Prevents `_syncCompassFromAdhan()` from overwriting forced alignment during dev mode
- **`?compass` URL param**: Auto-enables compass mode with forced alignment for desktop lookdev
