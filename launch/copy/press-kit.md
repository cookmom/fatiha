# A Gift of Time — Press Kit

## What It Is

**A Gift of Time** is a free, browser-based Islamic prayer clock built with Three.js. It provides accurate prayer times, a qibla compass, spatial adhan, and 11 beautifully crafted 3D clock dials — all running in a web browser with no installation required. It launches for Ramadan 2026 (February 18, 2026).

**URL:** [agiftoftime.app](https://agiftoftime.app)

---

## Key Features

| Feature | Description |
|---|---|
| **Prayer Times** | Accurate salah times calculated for the user's exact geolocation using established Islamic astronomical methods |
| **Qibla Compass** | Real-time great-circle bearing to the Kaaba in Mecca |
| **Spatial Adhan** | Immersive, spatially positioned call to prayer using Web Audio API |
| **11 Clock Dials** | Handcrafted 3D dial designs, each with unique aesthetic character |
| **Night Mode** | Soft, dark interface optimized for late-night prayers and Ramadan evenings |
| **Ramadan Fasting** | Suhoor/iftar countdowns and fasting progress visualization |
| **Offline Support** | PWA-enabled — works without an internet connection |
| **Privacy-First** | No tracking, no analytics, no data collection. Location stays on-device. |

## Tech Stack

- **Three.js** — 3D rendering engine for all clock dials and visual elements
- **Web Audio API** — Spatial audio positioning for adhan playback
- **JavaScript** — Fully client-side, no backend server
- **PWA** — Progressive Web App with offline capability
- **Astronomical calculations** — Solar position and Islamic prayer time algorithms
